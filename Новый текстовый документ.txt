Раздел языка TransacSQL используемый для управления данными в таблице. 
Это иснтрукция - Update, Delete, Insert, Select

SELECT - Инструкция для выборки данных, из одной или нескольких таблиц. 
	 Инструкция SELECT является наиболее часто используемой 
	 и емкой инструкцией в транзак SQL.

1. Предложение select включает выбираемые столбцы или поля из таблицы, а так же фукнции
	для получения сводных данных.
2. FROM - указывает на таблицы из которых выбирается данные с указанием связи между ними.
3. WHERE - Включает условия, которым должны удволитворять выбираемые данные.
4. GRUOP BY - Определяет поля по которым групируются данные для получения сводных данных
5. HAVING - Условия которые придявляются после группировки и вычисления результирующих данных 
6. ORDER BY -Указывает на поля по которым сортируются данные

При выполнении инструкции SELECT система выполняет следующую последовательность операций:
	1. По предложениям FROM SELECT и WHERE производоится выборка соответсующих полей из таблиц, удволитворяющих данному условию.
		Формируюется набор данных RS1
	2. Если существует предложение GROUP BY то из RS1 формируется RS2 в котором записи выстраюиваются по полям
		указанным в предложении GROUP BY 
	3. Зависимости от имеющейся функции в предложении select к группированным полям соответсующий вычисления. и вычисленные данные запписваются  4
		в RS3  в котором камбинация полей  не повторяются.
	4. Если Существует предложение HAVING то эти условия применяются к RS3 и удволитворяющие записи из RS3 записваются в RS4.
	5. Если есть предложени order by то записи RS4  упорядычиваются и записываются RS5

Агрегатные функции - AVG, CHECKSUM_AGG
